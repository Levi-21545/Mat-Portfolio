FUNÇÕES QUADRÁTICAS

SEMANA 1


O QUE É?

As Funções Quadráticas, também chamadas de Funções do 2º Grau, além de serem funções, como o próprio nome diz, tem algumas especificações a mais que a tornam um pouco diferentes das Funções Afim, ou também, Funções do 1º Grau. Podem ser definidas da seguinte forma:

f: ℝ → ℝ
f(x) = ax² + bx + c

Para que continue sendo uma Função do 2º Grau, o termo a precisa ser, necessariamente, diferente de zero, caso contrário, não teria influência na equação já que passaria a valer 0. Os gráficos de uma função quadrática sempre formam uma parábola, seja ela com a concavidade virada para os positivos ou para os negativos, e é no ponto y do gráfico onde fica esta curva, deixando ambos os lados iguais um ao outro.

RAÍZES REAIS OU ZEROS DA FUNÇÃO QUADRÁTICA.

As raízes da função são os valores para x onde f(x)=0. Quando representados no gráfico, são os pontos onde o eixo X as cruza. Para encontrá-las, é necessário atribuir 0 para y e resolver a equação de segundo grau obtida. Existem as seguintes formas de realizar esta equação:

  • Usando a Fórmula de Bháskara em qualquer equação de 2º grau.

    Para os que não estão lembrados, segue a famosa fórmula:

    x = (-b ± √∆)/2
    ∆ = b² − 4.a.c

    • Se ∆=0 a função tem duas raízes reais iguais;
    Se ∆>0 a função tem duas raízes reais diferentes;
    Se 0>∆ a função não tem raízes reais.

  • Por fatoração quando o coeficiente c é igual a 0.

    No exemplo f(x) = x² + 5x podemos resolver da seguinte forma:

    Para fatorar x² + 5x, precisamos tirar o fator comum de x de ambos os termos:
    x² + 5x = x(x + 5)
    x(x + 5) = 0
    x' = 0x + 5 = 0
    x'' = -5

  • Isolando o x quando a equação é incompleta com o coeficiente b é igual a 0.

    No exemplo f(x) = x² - 9 podemos resolver da seguinte forma:

    x² - 9 = 0
    x² = 9
    x' = -3x'' = 3

  • Pela soma e produto usando cálculo mental. É mais favorável usar desta maneira quando os coeficientes e as raízes são números inteiros e o a é igual a 1.

    A fórmula geral pode ser escrita como

    x1 + x2 = - ( b / a)
    x1 . x2 = c / a

    No exemplo f(x) = x² + 6x + 5 podemos resolver da seguinte forma:

    x² + 6x + 5 = 0
    x1 + x2 = -6x1 . x2 = 5
    -1 + (-5) = -6-1 . (-5) = 5
    x' = -6x'' = 5

Observações:
1) Uma função quadrática pode ter suas duas raízes, uma ou nenhuma raiz. Se ela tiver duas, a parábola irá cruzar duas vezes o eixo X. Agora, se tiver somente uma, cruzará o eixo X somente uma vez. Se não tiver nenhuma raiz, a parábola não cruzará o eixo X;
2) Muitas pessoas se confundem em achar que a função não existe quando resolvem a equação de Bháskara e na raiz resulta 0. Na verdade, ela existe, apenas tem raízes complexas.

Vértice ou Pontos de Máximo e Mínimo

O vértice é o ponto no gráfico no qual a parábola muda sua trajetória em y e começa a decrescer, caso antes crescendo, ou a crescer, caso antes decrescendo. Quando a>0, o vértice é também o ponto mínimo, e quando 0>a, o vértice seria o ponto máximo.

Maneiras de encontrar o vértice:

  • Encontrando as raízes.

    A abcissa X do vértice pode ser encontrada através da média aritmética entre as duas raízes, já que estará localizada bem no meio das duas. E a ordenada Y, logicamente, pode ser encontrada caso a gente substitua o X na função pelo valor encontrado antes.

    xv = (x' + x'') / 2
    f(xv) = yv

  • Usando a fórmula a partir dos coeficientes da função.

Construção do Gráfico

Diferente de como é feito com Funções Afim, onde apenas atribuímos valores quaisquer a X e marcamos os pontos no gráfico, com as Funções Quadráticas é necessária uma construção mais elaborada para uma melhor identificação. Ao determinarmos valores aleatórios, temos a chance de encontrar apenas uma parte da parábola, ou senão, uma simples reta, dependendo apenas da proporção utilizada. Para que isso não aconteça, é recomendado que sejam seguidos os seguintes passos:

  • 1º) Analisar a concavidade;
  • 2º) Calcular as raízes reais, se existirem;
  • 3º) Determinar o ponto de interseção com o eixo y;
  • 4º) Calcular as coordenadas do vértice.

Diferentes formas da Função Quadrática

Podemos escrever as Funções de 2º Grau das seguintes formas:

Forma Equação Função
Forma Geral ax² + bx + c = 0 f(x) = ax² + bx + c
Sendo o ponto (0,c) a interseção da parábola com o eixo y
Forma Canônica a(x - xv)² + yv = 0 f(x) = a(x - xv)² + yv
Sendo o ponto (yv, xv) o vértice da parábola.
Forma Fatorada a(x - x1)(x - x2) = 0 f(x) = a(x - x1)(x - x2) = 0
Sendo x1 e x2 as raízes.

E para convertê-las entre si, utilizamos estes métodos:

Para transformar: O que fazer?
Forma Canônica / Forma Fatorada → Forma Geral Realizar as multiplicações e potências, ou seja, expandir a expressão algébrica.
Forma Geral → Forma Canônica Processo de completar quadrados ou encontrar o vértice da parábola e posicionar os valores conforme o modelo genérico.
Forma Geral → Forma Fatorada Encontrar as raízes da função e posicionar os valores conforme o modelo genérico.
