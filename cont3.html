ENTENDENDO AS SEQUÊNCIAS

Sequências numéricas são sucessões de números que seguem um padrão específico. Elas podem ser finitas ou infinitas. Matematicamente é possível abordar aquelas em que se pode identificar o padrão de progressão e prever os valores de todas as posições. Olhe os exemplos abaixo:

1) 0, 4, 8, 12, 16, 20, 24, 28...
Regra: Adicionando 4 ao valor anterior.

2) 3, 8, 13, 18, 23, 28, 33, 38...
Regra: Adicionando 5 ao valor anterior.

3) 1, 6, 36, 216, 1.296, 7.776, 46.656...
Regra: Inicia no 1 e multiplica por 6 os termos seguintes.

Uma sequência pode ser uma função que associa a cada elemento n ∈ ℕ* um único elemento an e ∈ ℝ.

LEI DE FORMAÇÃO

Existem três formas de estabelecer suas formações:

  • Por fórmula de recorrência

    São dadas duas regras: uma para identificar o primeiro termo (a1) e outra para calcular cada termo (an) a partir de seu antecedente (an-1)

    6 primeiros termos da sequência definida por:
    (a1 = 2 | an = an-1 + 3, n ≥ 1)
    2, 5, 8, 11, 14, 17...

    6 primeiros termos da sequência definida por:
    (a1 = 1 | an = an-1 . 3, n ≥ 1)
    1, 3, 9, 27, 81, 243...

    Podemos também estabelecer a fórmula a partir de uma sequência dada.

    0, 2, 4, 6, 8, 10...
    (a1 = 0 | an = an-1 + 2)

    2, 6, 18, 54, 162...
    (a1 = 2 | an = an-1 . 3)

  • Expressando cada termo um função da sua posição n.

    É estabelecida uma fórmula que expressa an em função de n. Chamamos essa fórmula de Termo Geral.
    Vamos escrever os 6 primeiros termos da sequência infinita cujo termo geral é dado por an = 3n + 1, n ∈ ℕ*.

    n = 1 | a1 = 4
    n = 2 | a2 = 7
    n = 3 | a3 = 10
    n = 4 | a4 = 13
    n = 5 | a5 = 16
    n = 6 | a6 = 19

  • Por propriedade dos termos

    É informada a propriedade que os termos da sequência devem apresentar.

    6 primeiros termos da sequência infinita formada pelos números primeors positivos em ordem crescente.
    (2, 3, 5, 7, 11, 13...)
    Apesar de não existir nenhuma função ou regra para definir a sequência.

PROGRESSÃO ARITMÉTICA

Chama-se Progressão Aritmética (PA) a sequência de números em que cada termo, a partir do segundo, é a soma do anterior com uma constante r. São exemplos de PAs:

1) 1, 3, 5, 7, 9...
a1 = 1 e r = 2

2) 0, -2, -4, -6, -8...
a1 = 0 e r = -2

3) 4, 4, 4, 4, 4...
a1 = 4 e r = 0

Crescente: cada termo é maior que o anterior, ou seja, r > 0.
Decrescente: cada termo é menor que o anterior, ou seja, 0 > r.
Constante: cada termo é igual ao anterior, ou seja, r = 0.

Numa PA de razão r, o enésimo termo é dado por:

a1
a2 = a1 + r
a3 = a2 + r | a1 + r + r | a1 + 2r
a4 = a3 + r | a1 + 2r + r | a1 + 3r
...
an = a1 + (n - 1)r

Às vezes é mais adequado indicar o primeiro termo da PA como a0, deixando o termo geral como an = a0 + nr. Perceba que a9 = a6 + 3r, pois para ir de a6 para a9 nós avançamos 3 posições. Da mesma forma para retroceder posições, utilizando valores negativos multiplicando a razão.

INTERPOLAÇÃO ARITMÉTICA

Nas sequências finitas (a1, a2, ..., an) os termos a1 e an são chamados de extremos e os demais termos entre eles são chamados de meios. Interpolar, intercalar, inserir k meios aritméticos entre dois números a e b, significa obter uma PA de extremos a e b com n = k + 2 termos. Para encontrar esses termos temos que encontrar a razão. Vamos interpolar 5 meios aritméticos entre 3 e 6:

n = k + 2
n = 5 + 2 = 7

an = a1 + (n - 1)r
6 = 3 + (7 - 1)r
6 = 3 + 6r
3 = 6r
r = 0,5

PA: (3, 3,5, 4, 4,5, 5, 5,5, 6)

SOMA DOS TERMOS DE UMA PA

É possível somarmos todos os termos de uma PA finita sem somarmos um a um dos termos. Para isso, podemos utilizar da fórmula:

Sn = ( (a1 + an)n ) / 2

Calculando a soma dos 25 termos iniciais da PA (1, 7, 13...):

Descobrindo o 25º termo da PA usando a fórmula do termo geral:
an = a1 + (n - 1)r
a25 = 1 + (25 - 1)6
a25 = 1 + 24 . 6
a25 = 145

Agora sim podemos utilizar da fórmula da soma:
Sn = ( (a1 + an)n )/2
S25 = ( (1 + 145) . 25 ) / 2 = 1.825

Às vezes é mais adequado indicar o primeiro termo da PA como a0, deixando o termo geral como an = a0 + nr. Perceba que a9 = a6 + 3r, pois para ir de a6 para a9 nós avançamos 3 posições. Da mesma forma para retroceder posições, utilizando valores negativos multiplicando a razão.

PROGRESSÃO GEOMÉTRICA

Chama-se Progressão Geométrica (PG) a sequência de números em que cada termo, a partir do segundo, é o produto do anterior com uma constante 1. São exemplos de PGs:

1) 1, 2, 4, 8, 16...
a1 = 1 e q = 2

2) -1, -2, -4, -8, -16...
a1 = -1 e q = 2

3) 1, 1/3, 1/9, 1/27, 1/81...
a1 = 1 e r = 1/3

  • Crescente: cada termo é maior que o anterior.
    • PG com termos positivos: q > 1.
    • PG com termos negativos: 1 > q > 0.
  • Decrescente: cada termo é menor que o anterior, ou seja, 0 > r.
    • PG com termos positivos: 1 > q > 0.
    • PG com termos positivos: q > 1.
  • Constante: cada termo é igual ao anterior, ou seja, r = 0.
    • PG com termos nulos: a1 = 0 e q qualquer.
    • PG com termos iguais e não nulos: a ≠ 0 e q = 1.
  • Alternantes: cada termo tem sinal contrário ao do termo anterior. Isso ocorre quando 0 > q.
  • Estacionárias: apenas o primeiro termo é diferente de zero.

Numa PA de razão r, o enésimo termo é dado por:

a1
a2 = a1 . q
a3 = a2 . q | a1 . q . q | a1 . q²
a4 = a3 . q | a1 . q² . q | a1 . q³
...
an = a1 . qn-1

Às vezes é mais adequado indicar o primeiro termo da PA como a0, deixando o termo geral como an = a0qn. Perceba que a9 = a6 . q3, pois para ir de a6 para a9 nós avançamos 3 posições. Da mesma forma para retroceder posições, utilizando valores negativos multiplicando a razão.
